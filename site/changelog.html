Recent Highlights

Three releases that changed the product surface.

These are the recent milestones most likely to matter during evaluation, rollout planning, or upgrade review.

v0.56.0

Control-plane posture evidence and recovery-readiness visibility.

Derived backup cadence, observed backups, latest backup and checkpoint artifacts, restore readiness, and active/passive reference posture from live runtime state; surfaced it in Help & Docs production readiness; and aligned backlog, docs, OpenAPI, SDK metadata, Helm, OTLP, and website release surfaces on the new baseline.

v0.55.1

Release guardrails, a11y strictness, and Node baseline.

Aligned contributor and CI runtime expectations on Node 22, tightened release packaging and asset validation, hardened request-ID generation and install smoke coverage, and kept strict admin-console accessibility coverage in the release path.

v0.53.8

Session hardening, collector lifecycle, and release readiness.

Exchanged pasted console tokens for HttpOnly sessions, added collector lifecycle analytics with last-success/error history and failure streaks, introduced remediation change-review and recovery history, added a production demo lab, and aligned SDK, docs, OpenAPI, Helm, OTLP, and website metadata on the fresh release.
