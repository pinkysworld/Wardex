# Start Here

Recommended order:
1. Give the assistant `prompts/master-prompt.md`
2. Then give `prompts/phase-00-repo-bootstrap.md`
3. After each phase, run a review with `prompts/repair-this-phase.md` if needed
4. After phase 8 and beyond, use `prompts/production-hardening-review.md` periodically

Recommended operating rule:
- never ask the assistant to implement more than one phase at once
